Project Overview

WeCast is a european weather predictor which predicts the real time weather conditions of the european states. The day, date, weather type and the temperature is fetched from the backend and displayed as output.

WeCast uses 7timer API to fetch the real time data by having a seperate .csv file as a dataset which includes the latitude, longitude and the city name from which the day, date, time, weather comdition and the temperature is displayed as the final output.
